## Installation
1. Clone the repository
   ```bash
   git clone https://github.com/JibranMughal1/react-dashboard.git
   cd react-dashboard
   ```
2. Install dependencies
   ```bash
   npm install
   # or
   yarn install
   ```
3. Start the development server
   ```bash
   npm start
   # or
   yarn start
   ```
4. Build for production
   ```bash
   npm run build
   # or
   yarn build
   ```

## Demo
